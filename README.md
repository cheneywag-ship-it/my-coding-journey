### 1. 🟠 C 语言学习 (`01-c-language/`)
C 语言是理解计算机底层原理的基石。
- **`basics/`**: 存放基础语法练习代码（如：数据类型、控制流、指针、内存管理）。
- **`projects/`**: 基于 C 语言的小型实战项目（如：命令行工具、小游戏）。
- **`notes.md`**: 记录 C 语言的核心概念、易错点和学习心得。

### 2. 🔵 Python 学习 (`02-python-language/`)
Python 用于快速开发、脚本编写及数据处理。
- **`basics/`**: Python 基础语法、内置库的使用练习。
- **`scripts/`**: 日常自动化脚本、文件处理工具等实用代码。
- **`projects/`**: 综合性 Python 项目（如：Web 爬虫、数据分析可视化）。

### 3. 🧠 算法与数据结构 (`03-algorithms/`) *(规划中)*
通用的计算机科学基础，不局限于特定语言。
- **`leetcode/`**: LeetCode 刷题记录与题解。
- **`data-structures/`**: 常见数据结构（链表、树、图）的手写实现。
